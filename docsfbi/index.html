Tentang

FerbotInd adalah sebuah bot Telegram yang terbuat dari bahasa Python dan menggunakan Telethon. FerbotInd sudah berbahasa indonesia (belum sempurna dan masih akan dikembangkan lagi) memiliki kurang lebih 30 modul. Anda dapat menemukan bot ini dengan cara mencari Username @FerbotInd_bot atau dengan mengklik tombol.

Fitur

  • Fitur-fitur yang terdapat pada FerbotInd dapat anda lihat dengan cara mengetik /help Fitur antara lain sebagai berikut:
    • AFK.
    • Manajemen Grup dengan mudah.
    • Membatasi anggota spam.
    • Mebanned/Melarang orang jahat ke grup.
    • Menghapus pesan yang di blacklis.
    • Mengecek cuaca.
    • Menambahkan filter di grup.
    • Mengunci/membuka grup secara mudah.
    • Menyimpan catatan di grup.
    • Melakukan purge/menghapus pesan spam dengan mudah.
    • Menetapkan rules/peraturan grup.
    • Membuat stike.r
    • Translate dari bahasa ke bahasa lain.
    • Menetapkan pesan selamat datang/tinggal.

      Cara pakai

      Anda ingin memakai FerbotInd?. Cari saja usernamenya @FerbotInd_bot atau dengan link berikut FerbotInd

    • Pertama-tama anda dapat mencari username @FerbotInd_bot jika sudah anda dapat mengetik /start.
    • Jika sudah nanti akan ada tombol bantuan tekan saja, atau ketik /help.
    • Nanti akan ada list modul, tinggal anda pilh yang anda inginkan.

      Cara deploy

      Anda ingin membuat bot sendiri tanpa coding? dan yang pastinya GRATIS. Silahkan ikuti cara dibawah ini!

    • Pertama-tama silahkan anda membuat bot di BotFather
    • Jika sudah buatlah akun Heroku
    • Jika sudah buat juga akun Github
    • Selanjutnya silahkan buka akun github anda dan fork repository FerbotInd dan jangan lupa berikan star qaqa:)
    • Setelah itu buka kembali akun heroku anda, dan buat lah app baru.
    • Setelah membuat app, buka app tersebut lalu masuk ke Deploy yang terdapat di bagian atas.
    • Lalu klik tanda Github dan ikuti instruksi yang diberikan.
    • Setelah berhasil silahkan buka Settings/Pengaturan. Jika sudah klik Reveal Config Vars/Lihat konfig var.
    • Sekarang silahkan kumpulkan var yang dibutuhkan sebagai berikut:
      • MESSAGE_DUMP : Silahkan buat grup untuk log bot Anda. Jika sudah tambahkan FerbotInd disana dan ketik /id dan simpan id itu.
      • OWNER_ID : Dapatkan dengan cara balas pesan anda di grup dan ketik /id nanti akan ada id anda.
      • OWNER_USERNAME : Username telegram anda.
      • SPAMWATCH_API : Dapatkan di SpamWatchBot.
      • API_OPENWEATHER : Dapatkan di OpenWeather.
      • DATABASE_URL : Dapatkan dengan cara masuk ke akun heroku anda, lalu masuk ke app anda, setelah itu masuk ke Resources dan tambah add-on Heroku Postgres. jika sudah masuk ke akun postgresnya lalu ke settings/pengaturan lalu View Credentials, nanti akan ada URI catat urlnya.
      • GBAN_LOGS : Isi dengan id grup anda
      • LASTFM_API_KEY : Dapatkan di Last.fm.
      • MONGO_DB_URI : Dapatkan di Mongodb.com.
      • STRICT_GBAN : Isi True jika anda ingin Ban secara Global berfungsi
      • TOKEN : Isi dengan token yang anda dapatkan di BotFather.
      • WEBHOOK : Isi False
    • Isi semua config var dengan hasil yang telah didapatkan diatas.
    • Jika sudah semua silahkan aktifkan bot anda dengan cara Masuk ke app anda di heroku dan masuk ke Overview dan Configure Dynos lalu ketik tanda pensil dan klik saklar hingga berwarna biru lalu Confirm.
    • Yayyy bot anda sudah bisa dipakai jika anda mengikuti tutorial ini dengan benar. Silahkan pamerkan ke semua orang bahwa anda pencipta bot.

      Fungsi Var

      Berikut ini merupakan fingsi-fungsi dari Var sudah anda dapatkan dengan cara diatas.

    • API_OPENWEATHER : Api key yang berfingsi untuk mengecek cuaca.
    • DATABASE_URL : Digunakan untuk mneyimpan seluruh Notes, Filters. Pesan selamat datang/tinggal.
    • GBAN_LOGS : Untuk menampilkan info global ban jika ada yang user yang di gban
    • LASTFM_API_KEY : Api key yang berfungsi untuk Modul last.fm
    • MESSAGE_DUMP : Berfungsi untuk mengirim log eror maupun log berhasil diaktifkan ke grup yang anda buat.
    • MONGO_DB_URI : Digunakan untuk menyimpan daftar pengguna, daftar grup, status afk, blacklist, Global ban, dan data-data.
    • OWNER_ID : Digunakan untuk menjadikan siapa pemilik bot.
    • OWNER_USERNAME : Digunakan untuk mngeali username pemilik ke bot.
    • SPAMWATCH_API : Api key spamwatch yang berfungsi untuk modul spamwatch
    • STRICT_GBAN : Jika diisi True maka Global ban akan berjalan dan Jika False makan tidak akan berjalan.
    • TOKEN : Token bot anda yang digunakan untuk menjalankan bot anda.
    • WEBHOOK : Isi False jika anda tidak ingin menjalankan Webhook
